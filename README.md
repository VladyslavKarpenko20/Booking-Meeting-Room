# Booking Meeting Rooms API

A meeting room booking API with role-based authorization and PDF report generation

## Technologies

.NET 8
Entity Framework Core
PostgreSQL
QuestPDF
JWT
Swagger

## How to get started

1)Install .NET 8 SDK and PostgreSQL.
2)Clone the repository.
3)Configure the connection string in "appsettings.json".
4)Run the "dotnet ef database update" migration.
5)Run "dotnet run".
6)Open Swagger (the address will be shown in the console after launch).
